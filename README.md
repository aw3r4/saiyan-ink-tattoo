# saiyan-ink-tattoo
This is the official website of the Saiyan Ink Tattoo Salon in Hajdúnánás.
A Saiyan Ink Tattoo egy feltörekvő vállalkozás, amely Hajdúnánáson található. A tulajdonos egyben a tetováló is, akit az oldalon található elérhetőségeken kereshettek. A tökéletes tetoválások itt készülnek. Segítőkész és barátságos a tetoválónk garantáltan nem fogsz csalódni és egy profi tetoválással távozol innen.  

## Funkciók
- Reszponzív dizájn, működik különböző képernyőméreteken
- Hover animáció a név fölött (sárkány mozgás)
- Könnyen szerkeszthető HTML és CSS
- Tiszta, áttekinthető kód

## Használat
1. Klónozd a repót: [git clone https://github.com/aw3r4/saiyan-ink-tattoo.git](https://aw3r4.github.io/saiyan-ink-tattoo/)
2. Nyisd meg a `index.html`-t
3. Élvezd az oldalt
